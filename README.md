
# 2048 Game is a classic puzzle game implemented in JavaScript, HTML, and SCSS

## Project overview
- [DEMO LINK](https://zhyliaieva.github.io/js-2048-game)
- [Code](https://github.com/zhyliaieva/js-2048-game)

## How to play
- Use **arrow keys on keyboard** to move tiles
- Tiles with the same numbers merge when they collide
- The goal is to reach the **2048** tile
- The game ends when there are no free moves

## Controls
- **Arrow Left** — move left
- **Arrow Right** — move right
- **Arrow Up** — move up
- **Arrow Down** — move down
- **Restart** — restart the game

## Project structure
- `src/scripts/` — JavaScript game logic
- `src/styles/` — CSS/Sass styles
- `src/index.html` — main page

## Features:
- Merge tiles by sliding them up, down, left, or right.
- The score is automatically calculated as you combine tiles.
- You can restart the game at any time.
- Responsive design for different screen sizes.

## Using technologies
- `JavaScript`
- `HTML / CSS`
- `Sass`
- `GitHub / Git`

## Installation / How to run
1. Clone the repository:
```
git clone https://github.com/zhyliaieva/js-2048-game.git
```
2. Navigate to the project folder:
```
cd js-2048-game
```
3. Install dependencies:
```
npm install
```
4. Start the local development server:
```
npm start
```
5. Open http://localhost:3000 in your browser to view the page.
